# Sabores do Mundo

Em Jaboatão dos Guararapes, o empresário gastronômico **Diógenes Carvalho Matias** é o visionário por trás da bem-sucedida rede de restaurantes **"Sabores do Mundo"**.  
Com três filiais estrategicamente localizadas e uma base logística eficiente para abastecimento, Diógenes enfrenta o desafio de gerenciar múltiplos aspectos do seu negócio em expansão.  

Ciente da necessidade de uma gestão eficiente, Diógenes decide criar um **sistema abrangente** que integre o **controle de estoque**, a **gestão de funcionários** e o **monitoramento da produção** em suas cozinhas.  
O objetivo é **otimizar os processos**, **reduzir desperdícios** e **garantir a consistência** na oferta dos pratos que conquistaram o paladar exigente de seus clientes, onde todos os dados são salvos em um **banco de dados SQLite no Android**.
